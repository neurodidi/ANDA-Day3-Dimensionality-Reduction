## Session Overview

Dimensionality reduction methods were originally motivated by a central question in neuroscience: when a large population of neurons encodes a behavioral variable, such as movement direction, does each neuron represent that variable independently, or do neurons act collectively through a low-dimensional shared signal?

This unit introduces dimensionality reduction using different kinds of neural data. In the homework, you will explore how PCA can be used to extract features from spike waveforms recorded with a tetrode array. These features can then be clustered to identify individual units, a critical step in spike sorting algorithms.

The first in-person session will introduce PCA in more detail. Working with real spiking data and simulations, you will learn how to apply PCA, interpret its outputs, and distinguish neural responses to different stimuli in PCA space.

The final session will go beyond PCA to introduce more advanced dimensionality reduction methods, including probabilistic PCA (PPCA), factor analysis (FA), and Gaussian process factor analysis (GPFA).

The aim of this unit is not to provide a full mathematical treatment of all these methods. Rather, it is to give you the opportunity to explore different dimensionality reduction techniques and see how they can help us generate insights into the brain.
